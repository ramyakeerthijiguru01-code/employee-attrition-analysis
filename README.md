# Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition using SQL, Python and Power BI to identify factors influencing employee turnover.

## Tools Used
- SQL
- Python (Pandas, Matplotlib)
- Power BI

## Project Structure
- `data/` → dataset
- `sql/` → SQL queries
- `python/` → Python EDA script
- `powerbi/` → dashboard screenshot / Power BI file

## Key Analysis Performed
- Attrition count and percentage
- Department-wise attrition
- Salary analysis
- Experience analysis
- Performance analysis
- Age-group analysis

## Power BI Dashboard
The dashboard includes:
- KPI Cards
- Attrition by Department
- Attrition by Age Group
- Slicers for Department, Gender and Age Group

## Key Insights
- Certain departments show higher attrition
- Lower experience employees tend to leave more
- Salary and age group patterns help explain employee turnover
